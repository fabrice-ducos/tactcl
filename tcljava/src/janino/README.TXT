Updated to Janino 2.3.18. (2/28/2006)

This is an embedded Java compiler that will be used with
TJC to compile Tcl procs at runtime.

See src/org/codehaus/janino/doc-files/new_bsd_license.txt
for license text. Only the source code for the Janino
compiler was imported. Download the original zip if
you want the Javadoc HTML for the Janino classes.
